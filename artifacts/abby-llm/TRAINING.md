# Обучение Abby LLM

Abby — это **твоя собственная языковая модель**. Она не использует Ollama, OpenAI
или любые внешние API. Всё считается на твоём железе.

## Две модели — два пути

### 1. Встроенный «стартовый мозг» (работает прямо в приложении)

Внутри Abby работает **настоящая** статистическая языковая модель
(интерполированная n-gram: триграммы + биграммы + униграммы со сглаживанием).
Она реально обучается на твоих файлах **без GPU** прямо в приложении:

1. Открой раздел **Datasets**.
2. Перетащи файлы в окно (или в десктоп-версии нажми «Выбрать папку с диска»).
3. Перейди в **Training** и нажми **Запустить обучение**.
4. Следи за реальными метриками (loss, perplexity) на графике внизу.
5. В **Chat** задавай вопросы — Abby отвечает на основе выученных данных.

Это полноценная обучаемая модель, а не заглушка. Качество ответов растёт вместе
с объёмом данных. Подходит для быстрого старта и наборов до сотен МБ текста.

### 2. Полноценный трансформер для 67 ТБ (на твоём GPU)

Для корпуса в десятки терабайт нужен настоящий GPT-трансформер и видеокарта.
Скрипт `training/train_abby.py` — это реальный трейнер на PyTorch:

- архитектура GPT (multi-head attention, LayerNorm, GELU MLP);
- байтовый словарь (256) — работает с любым языком и кодом без токенайзера;
- **потоковое** чтение данных по файлам — корпус не грузится в память целиком,
  поэтому 67 ТБ обрабатываются по частям;
- чекпоинты каждые 2000 шагов.

```bash
# на машине с CUDA-GPU
pip install torch numpy
python training/train_abby.py --data /mnt/my-67tb-drive --out ./checkpoints --steps 100000

# несколько GPU
torchrun --nproc_per_node=8 training/train_abby.py --data /mnt/data --out ./ckpt
```

Параметры модели (`--n_layer`, `--n_head`, `--n_embd`) подбирай под объём VRAM.
По умолчанию ~124M параметров (как GPT-2). Для серьёзного качества на 67 ТБ
увеличивай размер и число шагов — это упирается в твоё железо, а не в Abby.

> Честно: обучить трансформер на 67 ТБ — это дни/недели работы GPU-кластера.
> Abby даёт тебе и мгновенный результат (встроенная модель), и полный код
> для большого обучения (этот скрипт). Никаких внешних сервисов.
